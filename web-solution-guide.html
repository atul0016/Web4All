Plan A — Traditional

WordPress + Hostinger

Best for clients who want to manage their own content without touching code. Fast to build, easy to hand over, industry standard.

WordPress Elementor Pro Hostinger Namecheap Domain Zoho Mail
01 Step-by-Step Setup
1

Buy Domain (Client Pays)

Go to namecheap.com — search and buy their domain. Example: riyaskitchen.com for ~$10/year. Always let client own this in their own account.

2

Buy Hosting on Hostinger (Client Pays)

Go to hostinger.com → Business Plan (~₹5000/year). This gives WordPress auto-installer, free SSL, email hosting, daily backups.

3

Connect Domain to Hostinger

In Namecheap, change the Nameservers to Hostinger's nameservers. Takes 5 minutes, live in 1–24 hours.

4

Install WordPress (1 click)

In Hostinger dashboard → WordPress → Auto Installer. Done in 2 minutes. SSL activates automatically.

5

Install Elementor + Your Theme

Install Elementor Pro ($59/year). Pick a starter template. Customize everything — colors, fonts, sections — with drag and drop. No code needed.

6

Build the Website

Home, About, Services, Gallery, Contact pages. Add WhatsApp button, Google Maps, contact form (WPForms). Setup SEO with Rank Math plugin.

7

Setup Business Email

Connect domain to zoho.com/mail — free up to 5 users. Client gets hello@theirbusiness.com for free.

8

Handover + Training

Record a 15-min Loom video showing client how to edit text, upload images, add blog posts. Share login credentials. They never need to call you for small stuff.

Plan B — Modern Custom

Next.js + Appwrite + Vercel

You build a custom admin panel. Client logs in, edits content. Changes reflect on site in ~1 min. Hosted free on Vercel. Zero monthly cost.

Next.js 14 Appwrite Cloud Vercel (Free) Namecheap Domain Tailwind CSS
01 How the Whole System Works
→

Client Opens Admin Panel

You give them a URL like yourdomain.com/admin. They log in with email + password (handled by Appwrite Auth — you don't build login from scratch).

→

Client Edits Content

They see a simple dashboard — Edit About Us, Update Menu, Change Phone Number, Upload New Photo. You decide exactly what they can touch. Nothing else is visible.

→

Data Saves to Appwrite

All content (text, images, data) is stored in Appwrite Database + Storage on Appwrite Cloud. Free up to 75k reads/writes per day.

→

Next.js Fetches + Rebuilds

Your Next.js frontend uses ISR (Incremental Static Regeneration) — it checks for new content every 60 seconds and rebuilds pages automatically. No manual deploy needed.

→

Visitor Sees Updated Site

Within 30 seconds to 2 minutes, any visitor will see the updated content. Completely automatic. Client never calls you for content changes.

02 Real Setup Steps (Dev Side)
1

Create Appwrite Cloud Account

Go to cloud.appwrite.io → Create free account → Create a new project. Note your Project ID and API key.

2

Setup Database Collections

In Appwrite → Databases → Create collections for each content type. Example: homepage, services, gallery, contact_info. Add attributes (fields) for each.

3

Setup Auth for Admin

In Appwrite → Auth → Create one user (the client). That's it. Appwrite handles login, sessions, tokens. You just call account.createEmailSession() in your code.

4

Build Next.js App

Run npx create-next-app@latest. Install Appwrite SDK: npm install appwrite. Build two parts — public website pages + protected /admin routes.

5

Connect Frontend to Appwrite

In your Next.js pages, use databases.getDocument() to fetch content. Use revalidate: 60 in your fetch for ISR. Data auto refreshes every 60 seconds.

6

Build Admin Panel UI

Simple forms in /admin page. Login form → on submit calls Appwrite auth. Edit forms → on save calls databases.updateDocument(). Use Tailwind for styling. Keep it simple.

7

Deploy to Vercel

Push code to GitHub. Go to vercel.com → Import repo → Add environment variables (Appwrite keys). Click Deploy. Done. Free forever for this scale.

8

Connect Custom Domain

In Vercel project settings → Domains → Add yourdomain.com. Point Namecheap DNS to Vercel's nameservers. SSL auto-activates. Done.
